Présentation du plug-in Geolimi

GeoLimi est un plug-in mit en place avec Qgis pour pouvoir intégrer, visualiser et modifier les données scientifiques du programme LIMITRACK.
Il est codé en Python et comporte une connexion directe avec une base de données Postgres.

Quelques informations

Cette page a pour but de vous aider dans la manipulation des données LIMITRACK via le plug-in Qgis mit en place. Elle reprend l'intégralité des fonctionnalités de l'outil.
Elle a été développée en HTML et est présente dans le menu "Aide" du plug-in.
La barre de navigation située au dessus de cette page correspond aux différentes parties du plug-in
